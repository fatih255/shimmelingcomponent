
Create Components that Shimmeling Effect

**Install Package**
    yarn add shimmelingcomponent

  


This Package using react-native-linear-gradient

    yarn add react-native-linear-gradient

 **Example**
 

    <Shimmering
		wrapperStyle={{
		width:  width  *  0.5,
		height:  25,
		borderRadius:  5}}
		/>
